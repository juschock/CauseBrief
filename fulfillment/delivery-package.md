# Delivery package structure

```text
CauseBrief_[CampaignName]/
├── README_How_to_Use.txt
├── 01_Email_Sequence.docx
├── 02_Social_Posts.docx
├── 03_Landing_Page_Copy.docx
├── 04_Flyer_Copy.docx
├── 05_Press_Release.docx
├── 06_Subject_Lines_and_CTAs.docx
└── 07_Posting_Schedule.pdf
```

## README_How_to_Use.txt (customer-facing)

```text
Start here.

1. Review the campaign details for accuracy.
2. Copy Email 1 into your email tool.
3. Schedule or post the social messages according to the suggested posting schedule.
4. Use the flyer copy in Canva or your print template.
5. Use the press release for local media, newsletters, or community calendars.
6. Adjust names, dates, links, or details before publishing.
```

Email to customer includes:
- ZIP download link (or attachment under size limits)
- Optional Google Doc folder link
- Reminder to verify dates/prices before publishing
- Support contact: causebrief@racoben.com
